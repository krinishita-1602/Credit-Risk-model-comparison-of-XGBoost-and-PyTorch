# Credit-Risk-model-comparison-of-XGBoost-and-PyTorch
1. The dataset used is the German Credit dataset loaded from UCI.
2. The code performs data preprocessing (one-hot encoding and scaling).
3. It builds a neural network using PyTorch.
4. It also trains a model using XGBoost.
5. Finally it compares metrics like accuracy, precision, recall, and F1-score.

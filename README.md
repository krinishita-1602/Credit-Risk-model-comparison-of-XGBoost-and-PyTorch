# Credit-Risk-model-comparison-of-XGBoost-and-PyTorch
This project compares machine learning approaches for credit risk prediction using the German Credit dataset. 
A neural network implemented with PyTorch is trained and evaluated against an XGBoost model to analyze differences in performance. 
The project includes data preprocessing, feature engineering, model training, and evaluation using metrics such as accuracy, precision, recall, and F1-score.
